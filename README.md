## Related papers

- **ICASSP 2024 (Part 1):** *Improving Oral Reading Fluency Assessment Through Sub-Sequence Matching of Acoustic Word Embeddings*   
- **ICASSP 2025 (Full / unified repo):** *A Unified Model for Oral Reading Fluency and Student Prosody*   

## What this repo contains

- Implementation of the **Part 1** method (ICASSP 2024), including the core sub-sequence matching logic operating over learned acoustic representations. 
- Additional code introduced in **Part 2** (ICASSP 2025) to form a more unified modeling approach spanning ORF and student prosody.
